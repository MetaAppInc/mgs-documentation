# MGS概览

##  什么是MGS
![](https://cdn.233xyx.com/1617072194133_503.png)
- MGS是233发起的好游戏深度合作计划，为游戏提供语音、好友关系赋能和更高的曝光导量；
- 与高dau的游戏互相成就，提供多维度游戏数据支持、商业优化和补贴。

## 对接MGS能获得什么
![](https://cdn.233xyx.com/1617072513774_410.png)
- 基础功能：登录、广告、充值；
- 增幅功能：233好友关系、免费语音；
- 获得曝光位置：首页联机大厅曝光位；
- 需要达标要求的吸量、留存和DAU时长数据；

## MGS游戏具有的特点
- 主要为联机互动玩法；
    - 需要和他人一起游戏，PVP或PVE玩法都可；
- 有友好的语音场景；
- 游戏玩法易懂难精；
